# 4.3 排序不等式

给定两个长度均为 n 的数组 a 和 b，可以交换同一数组内的元素，最小化/最大化

a[0]⋅b[0]+a[1]⋅b[1]+⋯+a[n−1]⋅b[n−1]
把 a 和 b 都从小到大排序，可以最大化上式。

把 a 从小到大排序，b 从大到小排序，可以最小化上式。
